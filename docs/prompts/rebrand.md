# Prompt: Rebrand the storefront

Copy this prompt into Claude / Codex / Cursor when you fork the template:

---

I'm rebranding this storefront. The new brand is:

- **Name:** `<your brand name>` (e.g. "Voltage")
- **Tagline:** `<one-line tagline>` (e.g. "The premium gaming toolkit")
- **Accent color:** `<hex value>` (e.g. `#7c3aed` for violet)
- **Vibe:** `<one or two adjectives>` (e.g. "premium and competitive")
- **Domain:** `<your-domain.com>`
- **Discord:** `<discord-invite-url>`

Apply the rebrand by editing `theme.config.ts` only. Use the recipe at `docs/recipes/change-brand-color.md` to pick the four accent variants. Don't touch any component files. Don't change the catalog yet — that comes next. After the edit, list every file you changed and briefly explain why.
